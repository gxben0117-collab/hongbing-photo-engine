# Red Bing Photo Travel Engine

這是一個純靜態的 AI 咒語產生器，分成三個主要工具頁與一個首頁入口。

## 頁面

- `index.html` - 入口頁與功能導覽
- `travel.html` - 寫真旅拍風格咒語產生器
- `magazine.html` - 雜誌棚拍風格咒語產生器
- `doll.html` - 公仔萌工作室

## 使用原則

- 介面顯示層維持繁體中文
- 邏輯層 key 使用英文
- 最終輸出咒語使用英文
- 需要對照時再查看 `docs/` 下的流程文件

本專案可直接部署到任意靜態主機，或使用 GitHub Pages 發佈。
開啟 `index.html` 即可在本機預覽。
