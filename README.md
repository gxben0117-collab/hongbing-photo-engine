# Red Bing Photo Travel Engine

This is a pure static prompt-tool site with four HTML entry points:

- `index.html` - home and navigation
- `travel.html` - travel photo prompt generator
- `magazine.html` - magazine shoot prompt generator
- `doll.html` - doll-style studio

## Usage

Open any HTML file directly in a browser. No install, no build step, and no external runtime dependency.

## Project layout

- `index.html` - landing page
- `travel.html` - travel workflow
- `magazine.html` - editorial workflow
- `doll.html` - doll workflow

## Maintenance notes

- This repo is a single-page static site set, so edits happen directly in the HTML files.
- Deploy the whole folder to any static host.
- `.gitignore` keeps OS and temporary files out of the repo.
