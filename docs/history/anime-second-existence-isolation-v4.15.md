# 動漫合鏡第二存在身份隔離：改前／改後

## 變更授權

Owner 於 2026-07-23 觀看實測結果後，同意依「幻想／雜誌」共用身份鎖定核心補強動漫合鏡頁。
此次不改變主角鎖臉優先序，而是新增第二存在不得重用主角身份的高優先級約束。

## 改前

- 高相似度模式已指定「Only one exposed human face」與守護者不可成為主角身份來源。
- 實測仍出現守護者以主角五官生成的情況，造成第二張肖像式人臉。

## 改後

- 新增「第二存在身份隔離」模組，緊接核心身份鎖定、臉部幾何與身份衝突處理之後。
- 明確禁止第二存在 duplicate、mirror、echo、clone 或 reuse 主角臉與身份。
- 第二存在限定為非人、頭盔、無臉、面具、抽象、靈體、機械、獸型或象徵能量形態。
- 保留關係敘事模組，使第二存在仍須由主角力量顯現並與主角互動。

## 影響

高相似度模式開啟時，生成順序為：主角身份核心 → 高相似度鎖臉 → 第二存在身份隔離 → 關係因果 → 其餘電影海報模組。
這避免將雜誌頁的「單人」規則錯誤套用到需要守護者的動漫合鏡頁，同時封堵第二張主角臉。
