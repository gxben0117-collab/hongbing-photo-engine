# 動漫合鏡畫風擴充與選項對應稽查：改前／改後

## 畫風擴充

在既有的寫實、漫畫、動畫、油畫與時尚插畫之外，新增水墨、水彩、厚塗、水粉、版畫、炭筆與粉彩。
每種畫風都有正向媒材證據與排他規則，避免只換色彩而沒有改變筆觸、線條、紙張或印刷風貌。

## 選項對應稽查

新增靜態稽查腳本，確認：

- 每個 radio 點選值都可找到非空白的咒語定義。
- 角色、服裝、力量、姿勢、背景、特效、鏡頭與媒材都有生成器串接。
- 關係模式、強度與視覺證據都有 `RELATION_DATA` 映射。
- 每個媒材都有渲染硬鎖定。
- 每個一鍵世界觀都有預設映射。
- 產生、複製與自訂文字欄位都有事件或咒語串接。
