# 動漫合鏡呈現媒材硬鎖定：改前／改後

## 問題

「最後呈現媒材」用來指定整張圖的繪畫與渲染風貌，例如漫畫墨線、動畫賽璐璐、油畫筆觸或真人攝影。
改前雖有媒材選項，但共用輸出規格仍包含商業攝影與電影攝影，造成非寫實媒材被沖淡。

## 修正

- 每個媒材增加高優先級的媒體鎖定與排他規則。
- 漫畫封面：墨線、網點、印刷質感，禁止照片與 3D 肖像。
- 時尚插畫：筆觸、馬克筆與色塊，禁止真人攝影。
- 油畫：厚塗、畫布、明暗法，禁止平滑數位照片。
- 動畫／網漫／美式漫畫：各自指定線條、上色與禁止的相鄰媒材。
- 寫實與黑白攝影仍保有相機、燈光與真實膚質規則。

## 優先順序

身份鎖定與第二存在身份隔離維持最高優先；媒材鎖定位於最終渲染輸出層，覆蓋共用的攝影輸出語言，不改變角色關係與臉部辨識規則。
