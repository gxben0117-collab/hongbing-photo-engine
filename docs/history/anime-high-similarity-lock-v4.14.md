# 動漫合鏡高相似度鎖臉模式：改前／改後

日期：2026-07-23  
適用頁面：`anime-hero.html`

## Owner 授權

Owner 於 2026-07-23 明確要求建立「核心不可動 + 高風險元素刪除／限制」的高相似度鎖臉模式，
並指定身份鎖定必須高於鏡頭、動作、服裝、變身、守護者、特效與背景。

## 改前

- 使用 fantasy 共用身份鎖定與臉部幾何保護。
- 關係敘事鎖定被前置至最終輸出開頭；雖可避免無關雙人海報，但不保證身份核心位於最高順序。
- 鏡頭可隨機選中純側臉、高角度、遠距主視覺或其他會縮小／遮蔽臉部的組合。
- 特效有「避免遮臉」描述，但沒有衝突時應降級元素的明確裁決句。

## 改後

最終高相似度輸出順序固定為：

1. 共用身份鎖定與臉部幾何鎖定
2. 高相似度衝突處理與五官淨空規則
3. 主角—第二存在關係鎖定
4. 真人骨架、光線、姿勢與其餘世界觀模組

新增規則：

- 身份保留高於所有鏡頭、姿勢、特效、服裝、變身、守護者與背景。
- 衝突時簡化、縮小或重新定位衝突元素，不得改臉。
- 臉部必須足夠大、清楚、不可被頭髮、面具、裝甲、粒子、能量、法陣或守護者遮擋。
- 只允許一張暴露的人類臉；守護者不可成為主角身份來源。
- 高相似度模式只保留平視、自然 3/4 與輕微低角度英雄鏡頭；隨機套用同樣遵守此限制。

## 影響

守護者、召喚、變身、機甲、式神、關係敘事和特效均保留；其角色改為服從主角身份可辨識度，
不再主導臉部角度或遮擋構圖。
