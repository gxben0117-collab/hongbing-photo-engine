# 寫真旅拍流程

## 流程

01. 隨機套用旅拍動感感覺（選用）
02. 選擇構圖模組
03. 選擇主風格
04. 選擇旅拍地點 / 主題
05. 選擇比例
06. 選擇鏡頭語感
07. 選擇動態感
08. 選擇輸出風格 / 畫面媒材
09. 生成咒語

## 隨機套用

- 「隨機套用｜旅拍動感感覺」會直接隨機抽選一組旅拍設定
- 會自動跳到生成結果
- 這個按鈕刻意避開黑金封面式的旅拍語感

## 主題類型

- 原本的真實地點維持地名式選項
- 新增的奇幻 / cosplay 類型使用「旅遊地點 + 造型企劃」
- 主題輸入可以自訂，介面只同步 chip 狀態，不應強制覆寫其他已選欄位

## 預設聯動

- `F 大景遠構` 會偏向更廣的鏡頭和流動感
- `法式慵懶` 會偏柔焦與靜態感
- `都會街拍` 會偏廣角與動態感
