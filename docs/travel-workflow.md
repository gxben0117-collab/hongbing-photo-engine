# 寫真旅拍流程

## 流程

01. 隨機套用旅拍動感感覺（選用）
02. 選擇構圖模組
03. 選擇主風格
04. 選擇旅拍地點 / 主題
05. 選擇比例
06. 選擇鏡頭語感
07. 選擇動態感
08. 選擇輸出風格 / 畫面媒材
09. 生成咒語

## 隨機套用

- 「隨機套用｜旅拍動感感覺」是**元素級獨立隨機**：構圖、風格、比例、鏡頭、光線、
  動態、媒材、姿勢、服裝、髮型、配件、主題文字，每一項各自從完整選項池獨立抽選，
  再動態組合成一份結果——不是從一批預寫好的完整組合裡三選一。
- 套用後會立即生成、顯示、捲動到結果。
- 這個按鈕刻意避開黑金封面式的旅拍語感（`style` 隨機池排除 `vogue`）。

## 生成 → 顯示 → 複製

- 改任一選項不會即時重算，要按「生成完整咒語」才會組裝並顯示。
- 生成過一次後，只要再改選項，輸出區會標記過期（金框提示、文字轉淡、複製鈕
  失效），必須重新按生成才能複製，避免複製到跟畫面選項對不上的舊咒語。
- 「一鍵套用｜XXX」與「隨機套用」按鈕套用後都會立即生成顯示，不用再手動按生成。

## 主題類型

- 原本的真實地點維持地名式選項
- 新增的奇幻 / cosplay 類型使用「旅遊地點 + 造型企劃」
- 主題輸入可以自訂，介面只同步 chip 狀態，不應強制覆寫其他已選欄位

## 預設聯動

- `F 大景遠構` 會偏向更廣的鏡頭和流動感
- `法式慵懶` 會偏柔焦與靜態感
- `都會街拍` 會偏廣角與動態感
