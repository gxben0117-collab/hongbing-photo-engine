# 雜誌棚拍流程

## 流程

01. 一鍵黑金高級封面（選用）
02. 選擇版面情境
03. 選擇主題 / 服裝方向
04. 選擇背景設定
05. 選擇姿態模組
06. 選擇人物入鏡比例
07. 選擇鏡頭語感
08. 選擇動態節奏
09. 選擇封面細節
10. 選擇圖片比例
11. 選擇輸出風格 / 畫面媒材
12. 生成咒語

## 預設聯動

- `黑金高訂` 會自動帶出窗光室內與黑金燈光
- `珠寶大片` 偏棚拍美妝與珠寶感
- `極簡高級` 偏乾淨背景與柔和光線
- `都會冷感` 偏硬邊輪廓與冷調光
- `深V高開衩` 與 `鏤空剪裁` 會走更強烈的封面語言
- `一鍵黑金高訂美妝特寫` 會自動切到臉部特寫入鏡比例

## 欄位順序原則

- `主題 / 服裝方向` 必須放在背景、姿態、鏡頭、光線之前
- 主題 chip 可以套用推薦背景與光線
- 使用者手動輸入主題時，只同步 chip highlight，不應再自動覆寫已選背景、鏡頭或光線

## 高風險系

- 只在需要更強時尚張力時使用
- 若模型容易失控，可先改用常規主題再逐步加強
